Part I: Object Oriented Programming Paradigms in C++:
  • UNIX Preliminaries, program format, compilation, running (Chap. 1,2)
  • Classes, File Processing, Preprocessor (Chap. 6, 14, 17)
  • Project 0: Data Types, Conditional Statements (Chap. 2)
  • Project 1: Functions, Arrays, Strings (Chap. 3, 4, 5)
  • Project 2: Operator Overloading (Chap. 8)
  • Project 3: Inheritance and Composition (Chap. 9)
Part II: Electrical Engineering Applications
  • Project 4: Numerical Solution of Linear Systems (also involves using MATLAB)
  • Project 5: Numerical Differentiation/Integration (also involves using MATLAB)
  • Project 6: Least Squares Approximation (also involves using MATLAB)
  • Project 7: Graph Connectivity Algorithms (Tentative)
  • Project 8: Graph Shortest Path Algorithms (Tentative)
  • Project 9: Implementation of LAZY-TEX Word Processor in C++ (Tentative)

Mandatory Projects: Each student is responsible for finishing all projects by his own.
